# Results Summary Component

## Overview
This project is part of the Codetribe Bootcamp assessment. It replicates the Frontend Mentor "Results Summary Component" challenge using **HTML, CSS, and JavaScript**.  
The component displays a score summary, dynamically loads data from a local JSON file, and adapts to different screen sizes.

## Built With

- Responsive layout for desktop and mobile
